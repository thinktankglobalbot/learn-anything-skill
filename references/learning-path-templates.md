# Learning Path Templates

Structured 4-week learning paths for popular topics. Each template includes goals, milestones, resources, and checkpoints.

---

## 1. Python Programming (Beginner → Intermediate)

| Component | Details |
|-----------|---------|
| **Learning Goal** | Write Python scripts confidently; understand core programming concepts; build a small project |
| **Prerequisites** | Basic computer literacy; no prior coding experience needed |
| **Hours/Week** | 5-7 hours |

### Weekly Milestones

| Week | Focus | What You'll Learn | Quiz Checkpoint |
|------|-------|-------------------|-----------------|
| Week 1 | Fundamentals | Variables, data types, strings, numbers, input/output, basic operators | Can you write a program that takes user input and performs calculations? |
| Week 2 | Control Flow | If/else statements, for loops, while loops, functions, scope | Can you write a function that processes a list and returns filtered results? |
| Week 3 | Data Structures | Lists, dictionaries, tuples, sets, list comprehensions | Can you manipulate nested data structures (e.g., list of dicts)? |
| Week 4 | Files & Modules | File I/O, error handling, importing modules, basic scripting | Can you write a script that reads a CSV file and summarizes the data? |

### Key Resources (Free)
1. **[Python.org Tutorial](https://docs.python.org/3/tutorial/)** — Official, thorough
2. **[Automate the Boring Stuff](https://automatetheboringstuff.com/)** — Practical, beginner-friendly
3. **[freeCodeCamp Python Course](https://www.freecodecamp.org/learn/scientific-computing-with-python/)** — Interactive
4. **[Exercism Python Track](https://exercism.org/tracks/python)** — Practice problems with mentorship
5. **[Real Python](https://realpython.com/)** — Deep-dive articles

## 2. JavaScript / React

## 3. Spanish (Conversational)

| Component | Details |
|-----------|---------|
| **Learning Goal** | Hold basic conversations; express needs, opinions, and past events; ~A2 level |
| **Prerequisites** | None |
| **Hours/Week** | 4-6 hours |

### Weekly Milestones

| Week | Focus | What You'll Learn | Quiz Checkpoint |
|------|-------|-------------------|-----------------|
| Week 1 | Foundations | Alphabet pronunciation, greetings, numbers 1-100, present tense of "ser" and "estar", basic sentence structure | Can you introduce yourself and ask/answer basic questions? |
| Week 2 | Daily Life | Common verbs (present tense), articles, gender/plural nouns, vocabulary: food, family, days, months | Can you describe your daily routine and family in simple sentences? |
| Week 3 | Questions & Past | Question words, "gustar" and similar verbs, simple past (preterite) of regular verbs | Can you ask for directions, order food, and describe what you did yesterday? |
| Week 4 | Conversational Flow | Irregular verbs, reflexive verbs, common expressions, filler words, practice dialogues | Can you have a 5-minute conversation about your week? |

### Key Resources (Free)
1. **[Duolingo Spanish](https://www.duolingo.com/)** — Gamified daily practice
2. **[SpanishDict](https://www.spanishdict.com/)** — Dictionary + grammar guides
3. **[BBC Mundo](https://www.bbc.com/mundo)** — Reading practice (news)
4. **[Language Transfer Complete Spanish](https://www.languagetransfer.org/)** — Audio course (excellent)
5. **[YouTube: SpanishDict Channel](https://www.youtube.com/spanishdict)** — Grammar videos

## 4. Machine Learning Basics

| Component | Details |
|-----------|---------|
| **Learning Goal** | Understand core ML concepts; build and evaluate simple models; know when to use what |
| **Prerequisites** | Basic Python; high school math (algebra, basic statistics) |
| **Hours/Week** | 6-8 hours |

### Weekly Milestones

| Week | Focus | What You'll Learn | Quiz Checkpoint |
|------|-------|-------------------|-----------------|
| Week 1 | ML Foundations | What is ML? Types (supervised, unsupervised, RL), basic terminology, train/test split, overfitting concept | Can you explain the difference between classification and regression? |
| Week 2 | Linear Models | Linear regression, logistic regression, gradient descent intuition, scikit-learn basics | Can you train a logistic regression model and interpret accuracy? |
| Week 3 | Trees & Evaluation | Decision trees, random forests, confusion matrix, precision/recall/F1, cross-validation | Can you compare two models and explain which is better (and why)? |
| Week 4 | Practical ML | Feature engineering basics, handling missing data, simple pipelines, build an end-to-end mini-project | Can you build a model that predicts X from a real dataset and explain your choices? |

### Key Resources (Free)
1. **[Google ML Crash Course](https://developers.google.com/machine-learning/crash-course)** — Practical, fast-paced
2. **[Kaggle Learn](https://www.kaggle.com/learn)** — Interactive notebooks
3. **[Scikit-learn Tutorials](https://scikit-learn.org/stable/tutorial/index.html)** — Hands-on with code
4. **[StatQuest YouTube](https://www.youtube.com/joshstarmer)** — Visual explanations
5. **[Hands-On ML Book (GitHub)](https://github.com/ageron/handson-ml3)** — Code-first learning

## 5. Financial Investing Fundamentals

| Component | Details |
|-----------|---------|
| **Learning Goal** | Understand investment vehicles, risk, diversification; build a simple investment plan |
| **Prerequisites** | None (basic math helpful) |
| **Hours/Week** | 3-5 hours |

### Weekly Milestones

| Week | Focus | What You'll Learn | Quiz Checkpoint |
|------|-------|-------------------|-----------------|
| Week 1 | Foundations | Time value of money, compound interest, inflation, risk vs return, asset classes overview | Can you calculate compound interest and explain why inflation matters? |
| Week 2 | Investment Vehicles | Stocks, bonds, ETFs, mutual funds, index funds; expense ratios, dividends | Can you compare an index fund vs actively managed fund and explain the trade-offs? |
| Week 3 | Portfolio Theory | Diversification, correlation, asset allocation, risk tolerance, rebalancing | Can you design a simple 3-fund portfolio for a specific risk profile? |
| Week 4 | Practical Planning | Tax-advantaged accounts (401k, IRA), dollar-cost averaging, avoiding common mistakes, building a personal plan | Can you outline a 10-year investment strategy for a specific goal? |

### Key Resources (Free)
1. **[Investopedia](https://www.investopedia.com/)** — Definitions, tutorials, simulator
2. **[Khan Academy Finance](https://www.khanacademy.org/college-careers-more/personal-finance)** — Core concepts
3. **[Bogleheads Wiki](https://www.bogleheads.org/wiki/Main_Page)** — Index investing philosophy
4. **[SEC Investor.gov](https://www.investor.gov/)** — Unbiased government resource
5. **[The Simple Path to Wealth (Book Summary)](https://jlcollinsnh.com/stock-series/)** — Simple, actionable

## Usage Notes

- **Adapt hours/week** based on your schedule—consistency matters more than intensity
- **Quiz yourself** at the end of each week; if you can't answer the checkpoint, review before moving on
- **Build projects** whenever possible—application beats passive consumption
- **Review spaced**: revisit Week 1 concepts in Weeks 2, 3, and 4 to improve retention
