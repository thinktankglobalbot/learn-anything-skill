# Topic-Specific Templates

Different topics need different approaches. Use these templates as starting points.

## Programming Topics

**Milestone structure:**
1. Setup & Hello World
2. Variables & Data Types
3. Control Flow
4. Functions
5. Data Structures
6. Projects (build something real)

**Resource types:**
- Official documentation
- Free online books
- Interactive tutorials (Codecademy, freeCodeCamp)
- Practice sites (Exercism, HackerRank, LeetCode)

**Practice focus:**
- Code EVERY day, even 15 min
- Build projects, not just read
- Debug intentionally â€” learn from errors

---

## Language Learning

**Milestone structure:**
1. Survival Phrases
2. Pronunciation & Alphabet
3. Basic Grammar
4. Vocabulary (high-frequency words)
5. Conversations
6. Cultural Context

**Resource types:**
- YouTube teachers
- Language exchange (HelloTalk, Tandem)
- Flashcard apps (Anki, Memrise)
- Native content (movies, music)

**Practice focus:**
- Speak from day 1 (even to yourself)
- Shadow native speakers
- Learn phrases, not just words
- Accept mistakes as progress

---

## Technical Skills (Trading, Design, etc.)

**Milestone structure:**
1. Fundamentals & Terminology
2. Tools & Platforms
3. Core Techniques
4. Analysis & Decision-Making
5. Risk Management
6. Real Practice (paper trading, mock projects)

**Resource types:**
- Industry blogs and news
- YouTube tutorials
- Courses (Udemy, Coursera)
- Communities (Reddit, Discord)

**Practice focus:**
- Paper trade / mock projects first
- Journal your decisions
- Review mistakes systematically
- Find a mentor or community

---

## Academic Subjects

**Milestone structure:**
1. Foundational Concepts
2. Key Theories/Frameworks
3. Problem-Solving Methods
4. Applications
5. Advanced Topics
6. Synthesis & Review

**Resource types:**
- Textbooks
- Lecture videos (YouTube, Coursera)
- Academic papers
- Practice problems

**Practice focus:**
- Feynman technique (teach it)
- Past papers / practice questions
- Concept maps for connections
- Spaced repetition for memorization

---

## Physical Skills (Sports, Instruments)

**Milestone structure:**
1. Fundamentals (grip, stance, posture)
2. Basic Techniques
3. Scales/Drills
4. Combinations/Flows
5. Performance/Application
6. Refinement

**Resource types:**
- Video tutorials
- In-person lessons (if possible)
- Apps with feedback
- Community classes

**Practice focus:**
- Deliberate practice (not just repetition)
- Slow is smooth, smooth is fast
- Record yourself for feedback
- Rest days matter for muscle memory

---
